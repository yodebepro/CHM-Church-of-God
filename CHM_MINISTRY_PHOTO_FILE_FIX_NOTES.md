# CHM Ministry Photo and File Fix

Preserves all existing filenames and content.

Updated `adm-ministries.html` so each existing ministry can:
- upload or replace its photo
- paste a direct photo URL
- upload a supporting file (PDF, Word, audio, video, image, presentation, or text file)
- paste a supporting file URL
- preview the current photo and open the current file
- save as draft or publish globally

Updated `chm-all-sections-sync.js` so published ministry records are matched to their existing public cards by ministry name. A published Prayer Ministry photo therefore replaces the Prayer Ministry icon/card media area rather than appearing in the first unrelated ministry card. Supporting files display as an Open Ministry File link.
