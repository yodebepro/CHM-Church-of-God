# CHM Ministry Duplicate Removal and Custom Sections Fix

This update preserves the existing ministry cards and filenames.

## Fixed
- Publishing a photo now replaces the icon/photo area of the matching ministry card.
- The same ministry is no longer copied again into a second list at the bottom.
- Obsolete generated ministry feeds at the bottom are removed.
- Existing ministry names, descriptions, categories, JSON import/export, draft, and global publishing remain intact.

## New options
Admin → Ministries now allows:
- typing a completely new category/section name
- adding an optional section subtitle
- adding a new ministry name not already in the existing list
- uploading a photo and supporting file for that new ministry

A genuinely new ministry is added once, inside its selected category. A new category is created only when the category name does not already exist.
