# CHM Hero Replacement, Volume, Mute, and Loop Fix

This update changes only the homepage hero-video workflow and preserves the complete previous baseline.

- Replacement links from Admin → About → Page Hero now become the public hero source.
- Google Drive file/view and docs/videos/play links are recognized.
- Added volume, start-muted, and loop controls.
- JSON export/import and Push to GitHub remain available.
- Newer GitHub JSON settings are preferred over stale browser data.

Direct/uploaded videos support exact volume and loop control. Browser autoplay rules can force sound off until a visitor interacts. Google Drive iframe fallback is controlled by Google and may limit exact volume or looping.
