# CHM Leadership, Sermon Library and Gallery Targeted Fix

This build preserves the previous baseline and changes only the requested public-page behavior.

- Elders and deacons are rendered only inside their own governance cards; unmatched published elder/deacon records create additional cards in the correct group instead of appearing at the bottom.
- Sermon series buttons, category selector, format selector and search now filter the library. Published sermon media opens in an actual YouTube/Vimeo/video/audio player.
- Published gallery images replace the matching public category placeholders and additional images remain inside the gallery grid.
- Existing filenames, admin workflows, JSON files and all other previously completed features remain unchanged.
