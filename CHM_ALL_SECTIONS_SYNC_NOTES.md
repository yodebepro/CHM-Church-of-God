# CHM All-Sections Global Sync Update

This update preserves every existing filename and layout while extending the leadership-style admin-to-public behavior to the other CMS sections.

Public-facing sections now read published records and media into their proper cards: Announcements, Events, Sermons, Gallery, Departments, Leadership, Teams, Ministries, Locations, About/Page Content, Navigation, Settings/Colors, and the homepage hero configuration.

Private administrative records—Members, Prayer Requests, Messages, and Giving Records—are saved and exported globally but remain private to Admin rather than being displayed publicly.

The new compatibility file is `chm-all-sections-sync.js`. Existing files were retained under their original names.
