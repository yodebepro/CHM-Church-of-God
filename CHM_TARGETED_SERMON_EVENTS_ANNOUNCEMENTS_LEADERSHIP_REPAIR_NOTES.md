# CHM Targeted Content Repair

This build preserves the uploaded CHM website and changes only the requested areas.

- Sermon uploads now use the existing GitHub media uploader, save video/audio URL aliases, publish `site-data.json` and `chm-sermons.json`, and render YouTube, Vimeo, direct video, and audio on the public Sermons page.
- Events now save status changes before publishing and render published Upcoming, Recurring, and Past records in the correct public tabs.
- Announcements now save Current, Pinned, and Archived states and render each group in its correct public tab.
- Homepage leadership photos are matched strictly by slot: Senior Pastor, Executive Pastor, Worship Director, Outreach Director.
- About-page servant-leader photos are matched strictly by slot: Senior Pastor, First Lady, Associate Pastor, Youth Pastor.
- All other files, layouts, hero-video features, ministry fixes, GitHub settings, JSON workflows, and prior protections remain unchanged.
